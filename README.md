```bash
cd P2PChat
```
to start type:
```bash
expo start
```
in terminal. Or then one of:
```bash
yarn android
yarn ios
yarn web
```

## Installing NPM and Expo

This requires npm and expo-cli. To get that [install node JS and npm](https://nodejs.org/en/download/)
and type:
```bash
npm install -g expo-cli
```
